Computer Vision Projects

Various computer vision projects showcasing different applications and techniques.

Food Ingredient Detection

A user could scan images of grocery items in the system, and the neural network predicted food recipes that can be made using those ingredients. Develop a 3 layer shallow CNN network for the detection of food ingredient using MobileNetV2 model as a pre trained weights. Integrate model in the flask.

Pig Unloading Detection

Train Yolo V11 detection model on the customized data set for the detection of Human, Pig and Paddle.

Urban Planning System

Used Yolo model for detection of cars and bikes for traffic management, urban surveillance and safety measures with customized data.

Construction site security compliance system

A vision (CNN) based system to ensure safety compliance by detecting people having inadequate safety equipment like helmets, and tools.

Fire detection

Detect fire in warehouses by processing CCTV footage using vision algorithms.
