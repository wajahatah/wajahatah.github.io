Responsibilities

Developed and optimized computer vision algorithms for pose detection of multiple individuals in a frame, leveraging normalized datasets and advanced preprocessing techniques. Implemented machine learning models for classification, utilizing Albumentations (v1.4.20) and MMCV-full (v1.7.2) for data augmentation and training. Managed 16 RTSP cameras for seamless real-time processing in production environments. Conducted rigorous testing and evaluation using 35 training videos to validate model performance.
