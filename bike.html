Electric Bike

Development of Electric Vehicle system for indiginize Electric Bike include development of bike, battery, and charging stations and their IoT based circuits for easy access and management mobile application and web based dashboard development

Key Responsibilities

  • Firmware development of ESP8266 and ESP32 for bike and battery IoT
  • Firmware development of charging stations system using ESP NOW protocol for Master Slave system
  • Implemented security features of ESP32 and ESP8266 includes secure boot and firmware encryption along with secure OTA (Over the Air) features
  • Developed testing and data acquisition system of Lithium Ion batteries
  • Prepare python script for data analysis and machine learning algorithm for battery degradation prediction
